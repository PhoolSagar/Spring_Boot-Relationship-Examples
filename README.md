# Spring Data JPA Relationships Practice 🚀

This repository contains multiple Spring Boot practice projects focused on understanding and implementing different **Spring Data JPA entity relationships**.

These projects were developed as part of backend development learning using **Java 21 (LTS)** and Spring Boot.

---

## 🛠 Technologies Used

- Java 21 (LTS)
- Spring Boot
- Spring Data JPA
- Hibernate
- MySQL
- Maven

---

## 🎯 Learning Objectives

- Understand how JPA relationship mapping works
- Implement One-to-One relationships
- Implement One-to-Many and Many-to-One relationships
- Implement Many-to-Many relationships
- Work with `@JoinColumn` and `@JoinTable`
- Understand Cascade types
- Learn Fetch strategies (EAGER & LAZY)
- Strengthen relational database fundamentals

---

## ▶️ How to Run

1. Clone the repository  
2. Open any project in Eclipse / IntelliJ / STS  
3. Set Project SDK to **Java 21 (LTS)**  
4. Configure database credentials in `application.properties`  
5. Run as Spring Boot Application  
6. Verify generated tables and relationships in MySQL
